Text Styles

H1: Main Page Heading(Title Page)

tip: h1 should only have one header this style per page.

  • Font-weight: 700 (bold)
  • Font-size: 26px
  • font-family: Merriweather

H2: Heading

tip: This should be to identify major sections you want the user to immediately be drawn towards.

  • Font-weight: 600 (bold)
  • Font-size: 20px
  • font-family: Merriweather

H3: Subheading

tip: This should be used for smaller focal points, such as articles, that you want the user to notice when scanning the page.

  • Font-weight: 500
  • Font-size: 18px
  • font-family: Merriweather

P: Paragraph text

tip: This should be used for smaller focal points, such as articles, that you want the user to notice when scanning the page.

  • Font-weight: 400 (Regular)
  • Font-size: 14px
  • font-family: Merriweather
